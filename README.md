# Online JSON Schema Validator

JSON Schema 校验工具专题项目，用于根据 JSON Schema 校验 JSON 数据结构，适合接口契约校验、配置检查和测试数据验证。

## Online Tool

[JSON Schema校验工具](https://www.abplus.xin/tools/json-schema-validator)

## Features

- JSON Schema 校验
- JSON 数据结构验证
- 接口契约检查
- 配置字段和类型检查

## Use Cases

- 校验接口响应是否符合 Schema
- 检查配置 JSON 字段类型
- 为测试数据生成校验规则
- API 文档和契约测试前验证数据结构
